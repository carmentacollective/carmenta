# AI Team

Agents that work alongside humans to manage knowledge, assist with tasks, and enhance
the Carmenta experience.

All agent prompts follow the prompt engineering standards:

@.cursor/rules/prompt-engineering.mdc

Key principles for agent prompts:

- Goal-focused, not step-prescriptive
- Trust the model to determine implementation
- Focus on WHAT (outcomes) not HOW (process)
- Clear structural delimiters using XML when needed
- Consistent patterns across examples
